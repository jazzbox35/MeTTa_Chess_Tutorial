\subsection{MettaWamJam}

Installing MeTTa on your own machine is easy and it will likely run a bit faster!

Check out the MettaWamJam server. You can use the available Docker images to instantly set up your own MeTTa server. Or, you can clone the repo and customize your own server as you need. This allows you to load your own
atomspace and run queries and updates against it.

<a href="https://github.com/jazzbox35/MettaWamJam" target="_blank" rel="noreferrer"><b>MettaWamJam Server</b</a>

"Wam" means Warren Abstract Machine and "Jam" means it is a toolbox of AI components.
